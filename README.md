# System Architecture
Farmer uploads or captures a crop leaf image
Image is sent to the Flask backend
Backend preprocesses the image
The trained TensorFlow model predicts the disease
System calculates:
Disease name
Confidence percentage
Recommended treatment
Estimated medicine cost per acre
Results are displayed on the web interface

# How It Works
Open the website
Go to Scan Crop
Upload or capture a leaf image
Click Detect Disease
The system shows:
Disease Name
Confidence %
Recommended Treatment
Medicine Cost per Acre
